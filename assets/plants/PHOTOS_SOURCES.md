# Источники фото

## Свои снимки (id 101–105)

Фото из JPG в корне проекта, нормализованы локально.

## Wikimedia Commons (скрипт `photos:commons`)

Для id 1–100 можно подобрать фото автоматически:

```bash
npm run photos:commons
```

- Ищет по латинскому названию вида.
- Берёт только файлы с лицензией **CC** или **Public domain**.
- Сохраняет `assets/plants/{id}.webp` и `attribution.json` (автор, лицензия, ссылка).

На странице каталога при публикации в интернет желательно указать: «Фото: Wikimedia Commons, лицензии CC / PD» и ссылку на `attribution.json`.

## Ограничения

- Не все 100 видов имеют удачный кадр «растение в горшке на белом фоне».
- Часть снимков — в саду/ботсаду; скрипт всё равно приводит к белому полю, но не заменяет студийную съёмку.
- При ошибке `429` подождите и запустите снова: `node scripts/fetch-commons-photos.js --ids 3,6,7`
