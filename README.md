# PlantFit Pro

Локальный веб-проект для интеллектуального подбора растений под реальные условия помещения.

## Что внутри

- `index.html` — приветственный лендинг проекта.
- `plant_selector_catalog_v6_photos_lux_fixed.html` — основное приложение-каталог с рекомендациями.
- `assets/css/styles.css` — стили лендинга.
- `assets/js/landing.js` — логика лендинга и быстрый переход в каталог.
- `assets/plants/*.webp` — фото растений (формат 4:3, белый фон, растение целиком).
- `scripts/normalize-plant-photos.js` — пакетная нормализация фото из каталога.

## Функционал

- Подбор растений по свету, влажности, температуре и уровню ухода.
- Фильтры и сортировка каталога.
- Система избранного и сравнение растений.
- Экспорт подборки и аудит данных каталога.

## Быстрый старт

1. Откройте `index.html` в браузере.
2. Нажмите `Начать подбор` или используйте `Быстрый старт`.
3. Перейдите в каталог и настройте фильтры под свои условия.

### Фото растений

После замены исходников в `RAW_PLANTS` (поле `photo` как data URI) можно снова привести все снимки к единому виду:

```bash
npm run photos:normalize
```

Скрипт сохраняет `assets/plants/{id}.webp` и подставляет пути в каталог. Исходный цветной фон на фотографии скрипт не вырезает — только белое поле и вписывание растения целиком. Для «студийных» кадров без фона нужны отдельные исходники или ручная обработка.
